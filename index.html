Rangkuman Git

Setelah repositori diinisialisasi dan informasi pengguna diatur, Anda dapat mulai menambahkan file baru ke repositori. Misalnya, dengan membuat file README.md dan menambahkannya ke staging area menggunakan git add README.md. Setelah itu, lakukan commit terhadap perubahan tersebut dengan menggunakan perintah git commit -m "Pesan commit" untuk menyimpan perubahan secara permanen dalam repositori. Jika Anda ingin berbagi proyek Anda dengan orang lain, Anda dapat menambahkan remote repository dengan perintah git remote add origin dan mengirimkan perubahan ke remote repository tersebut menggunakan git push -u origin master.
